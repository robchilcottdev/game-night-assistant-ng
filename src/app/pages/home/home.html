Welcome to Game Night Assistant! Pick a game below to start keeping score. If you want to trigger your Home Assistant scripts based on game events (like when a certain score is reached, a player is eliminated or a game-specific event happens), click the Script settings button at the bottom of the screen to choose your scripts.

Generic Score Keeper

Records a list of players and their scores. Trigger options include running scripts when a certain score is reached, when a player is removed from the game, and when a player scores zero. Ideal for general score-keeping and helping to run quiz night.

Farkle

Throw six dice, aim for 1s, 5s, or scoring combinations... and keep rolling for as long as you dare! Let Home Assistant scripts celebrate moments like 6-of-a-kind and three-pairs, whilst commiserating over dreaded farkles.

Skull King

Score pad for the trick-taking game Skull King, me hearties. Home Assistant triggers include the Skull King making an appearance, pirates capturing mermaids, mermaids capturing the Skull King, and so forth.
